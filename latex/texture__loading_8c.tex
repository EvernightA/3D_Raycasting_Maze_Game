Load XPM Textures -\/ Getting Your Wall Pictures Ready! üñºÔ∏è 

Imagine you have a bunch of wallpaper rolls for your room. Before you can put them up, you need to unroll them and get them ready. That\textquotesingle{}s what this function does for your game walls!

It takes the picture files you specified (north, south, east, west walls) and loads them into the computer\textquotesingle{}s memory so the game can use them. Each picture becomes a special "{}texture"{} that the computer can show on walls.

If any picture can\textquotesingle{}t be loaded (like if the file is missing), it tells you there\textquotesingle{}s a problem and stops the game.
