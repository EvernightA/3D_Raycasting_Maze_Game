DDA (Digital Differential Analyzer) Algorithm Implementation üå± 

This file contains functions for implementing the DDA line drawing algorithm, which is used in raycasting to determine wall intersections. Think of it as walking through a garden grid to find where you hit a flower bed (wall)!

DDA Line Drawing -\/ Like Walking in a Grid Garden! üå± 

Imagine you\textquotesingle{}re in a big garden divided into square patches. You want to walk from one flower (start point) to another flower (end point). But you can only walk along the edges of the patches, like a grid.

DDA (Digital Differential Analyzer) is like planning your walk\+:
\begin{DoxyItemize}
\item It figures out how many steps you need to take
\item It calculates how much you move left/right (x\+\_\+inc) and up/down (y\+\_\+inc) each step
\item It\textquotesingle{}s like saying "{}for every 2 steps right, I need to take 1 step up"{}
\end{DoxyItemize}

This is the main walking function! You start at your first flower and take small steps until you reach the second flower. At each step, you\+:
\begin{DoxyEnumerate}
\item Remember where you are (update\+\_\+current)
\item Add a new point to your path (like leaving breadcrumbs)
\item Move a little bit left/right and up/down
\item Count your step
\end{DoxyEnumerate}
